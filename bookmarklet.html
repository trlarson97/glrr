GLRR GRAB — INSTALL

This adds a one-tap button to your browser. On an athletic.net results or rankings page, just tap GLRR Grab (no need to select anything) — it opens the generator already filled in. A rankings page makes one graphic; a full meet page lists every event so you can make them all at once.

Generator it points to:

iPhone / Safari

  1. Tap Copy bookmarklet code below.
  2. Bookmark this page: tap Share → Add Bookmark → Save.
  3. Open Bookmarks (book icon) → Edit → tap that new bookmark.
  4. Rename it GLRR Grab, and in the address/URL field, delete what's there and paste the copied code. Save.
  5. Done. On any athletic.net results page: Bookmarks → GLRR Grab.

Desktop (Chrome / Edge / Safari)

  1. Show your bookmarks bar (Ctrl/Cmd-Shift-B).
  2. Drag this button onto the bookmarks bar:
📥 GLRR Grab
Tip: just tap GLRR Grab with nothing selected — it grabs the whole page and the generator figures out the events automatically. For a very large meet, if the link ever looks cut off, you can instead copy the page and paste it into the generator's box by hand — same result, no length limit.
